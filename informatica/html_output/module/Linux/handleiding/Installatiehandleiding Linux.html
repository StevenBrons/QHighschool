Voorbereiding

  1. Zet VT-x (Intel) of AMD-v (AMD) aan in de BIOS voor virtualisatie > ondersteuning.
    >
    > (Als je dit hebt maakt het het uitvoeren van de virtual machine > een stuk gemakkelijker voor je pc. Als je dit niet hebt geen > probleem.) >

  2. Download de Ubuntu Desktop LTS (Long-Term-Support) ISO van > https://ubuntu.com/download/desktop

  3. Download en installeer VirtualBox van > https://www.virtualbox.org/wiki/Downloads, > kies hierbij onder VirtualBox platform packages jouw > besturingssysteem.

  4. Zorg dat je ten minste 20 GB vrij hebt op je harde schijf.

  5. Maak een virtual machine voor Ubuntu.

    1. Maak in VirtualBox een nieuwe virtual machine door op New te > klikken. Vul bij Name “Ubuntu” in (zonder > aanhalingstekens).

    2. Geef de virtual machine 2048 MB aan RAM. Doe dit met de > slider of door het in te vullen.

    3. Klik op Next tot je aankomt bij File location and size. > Geef de virtual machine 20 GB aan opslag, hierdoor kan je > vrij testen.

    4. Zorg dat Ubuntu geselecteerd is. Druk dan op Settings. > Ga naar het tabblad System en dan Processor. Zorg dat > je 2 Processor(s) hebt toegekend.

    5. Ga nu naar Storage en kies de CD-ROM waar Empty bij > staat. Klik rechts van IDE Second Master en dan Choose > Virtual Optical Disk file. Kies nu de ISO die je hebt > gedownload in stap 2.

    6. Druk op OK.

Installatie

  1. Start de virtual machine.

  2. Kies je taal (de verdere stappen zijn met Engels gedaan), klik > daarna op Install Ubuntu.

  3. Kies je installatie taal.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  4. Kies de keyboard layout. De meeste laptops zullen English (US) > hebben.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  5. Kies voor de Normal Installation en vink beide checkboxes aan.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  6. De virtuele schijf van 20 GB staat nog niks op. Daarom is het het > makkelijkst om hier Erase disk and install Ubuntu te kiezen. > Er wordt hier niets van je echte harde schijf verwijderd.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  7. Klik op Continue
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  8. Klik op Continue bij de tijdzone.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  9. Vul je gegevens in zoals je ze wil hebben en klik op Continue.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  10. Wacht totdat Ubuntu klaar is met installeren, klik daarna op > Restart Now.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}

  11. Druk op ENTER wanneer gevraagd.

  12. Log in en je bent klaar om Ubuntu te gebruiken.
    > {width="6.270833333333333in" > height="3.9166666666666665in"}
