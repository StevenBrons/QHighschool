Spelregels
Inleveren van een Python Opdracht

  • Je mag zo vaak als je wil je Python-code aanbieden en om feedback vragen.
    Daar is één regel voor: je moet aangeven waar je precies feedback op wil door het stellen van een specifieke vraag.
    Een voorbeeldvraag: waarom is de for-loop van regel 65 niet correct?

  • Ik heb twee dagen de tijd nodig om je van commentaar te voorzien.

  • Hoe kun je feedback vragen? Stel je vraag in een privechat via de Teams app. Stuur tegelijk je .py-bestand mee als bijlage in de chat. Bij een ontbrekend .py-bestand, kan ik je daar natuurlijk geen feedback op geven.

  • Wanneer je nul of meerdere keren feedback hebt ontvangen en je wil inleveren: Gebruik de juiste opdrachtmodule van de Teams app ‘Assignments’. Voeg je .py-bestand toe als bijlage.

  • Denk er aan dat je je juiste .py-bestand bij de juiste opdracht invoegt!
